## Scripts
---

#### Python

* 2048.py

Script que executa automaticamente o jogo 2048

* arquivos.py

Script para backup de arquivos com um termo especifico

Uso:
```sh
$ arquivos.py /diretorio/de/pesquisa

```

* pass.py

Script para criação de passwords

Uso:
```sh
$ pass.py <quantidade de caracteres>

```
